Identification of correlated mutations in Influenza using a network analysis approach

This application provides a simple interface to analyze/understand results of network analysis of correlated mutations in Influenza. Residues in each protein act as nodes in the graph and edges in the graph are created based on inter-residue correlated mutations. Our approach used ‘maximal information coefficient’ (mic) to compute correlations between residues and we create edges between nodes if mic exceeds a cut-off. We studied the properties of the system for different mic cut-off values. We constructed graphs for multiple Influenza datasets.
